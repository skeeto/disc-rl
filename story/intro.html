Welcome to Disc RL

Disc RL takes place in the Tron universe. You are a program trapped in a hostile, infected computer. You need to find the Input/Output Tower so that you can re-establish contact with your User. As you travel deeper into the internals of this system you will encounter increasing levels of corruption.

Your game is automatically saved locally, so it is always safe to close this tab. You can visit this page again later to continue your game. But remember, death is permanent.

Identity Disk

Every program has its own identity disc. It can be used to strike other programs or to block attacks (armor). It can also be thrown at enemies, automatically returning to its owner. However, when thrown, the disc cannot be used to block attacks for a short period of time!

Movement is with the numpad or like NetHack. Use the hjkl keys to move left, down, up, and right. Use yubn to travel the diagonals.

Use Q to quit this game and start with a new character.

Browser Support

Disc RL is known to work properly in Firefox, Chrome, and Safari. The game's source code is in the public domain and can be found on GitHub.
